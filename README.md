# Asteroids (Mobile-Friendly) — GitHub Pages Ready

A fast, simple Asteroids-style clone built with **HTML5 Canvas**. Works on desktop + mobile.

## Controls
- **Mobile**: Buttons at the bottom (rotate left/right, thrust, shoot)
- **Keyboard**:
  - Left/Right or A/D = rotate
  - Up or W = thrust
  - Space / Enter = shoot
  - P = pause
  - R = restart

## Deploy to GitHub Pages
1. Create a new repo (or use an existing one).
2. Upload all files in this zip to the **repo root**.
3. GitHub → **Settings** → **Pages**
4. Source: **Deploy from a branch**
5. Branch: **main** / folder: **/(root)**
6. Save — your game will be live at:
   `https://YOUR_USERNAME.github.io/YOUR_REPO/`

## Notes
- No build step, no dependencies.
- Everything runs client-side.
